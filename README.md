## Run

To run the project project, clone the repo and follow the stepts presented below:

### Build

To build:

```
docker-compose build
```

### Run 


To run:


```
docker-compose up
```

### Unit Test

To run the unit tests:

```
docker exec -it vue-js yarn test:unit
```
## Requirements

* Docker 
* Docker compose
